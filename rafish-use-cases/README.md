# RaFish use cases

This directory connects imported AI material with practical RaFish work.

Use it before promoting anything to `active/`.

## Areas

- `prestashop/` — store, modules, SQL, imports, order workflows.
- `allegro-baselinker/` — marketplace and shipping automations.
- `gmail-drive/` — Gmail, Google Apps Script, Drive, invoice flows.
- `grafiki-etykiety/` — labels, packaging, image prompts, brand consistency.
- `marketing-opisy/` — product descriptions, SEO, Allegro copy, campaigns.
- `obsluga-klienta/` — complaints, reviews, messages, service scripts.
- `research-hurtownie/` — suppliers, products, scraping, comparison research.

Each subfolder tracks typical tasks, useful sources, candidates for `active/`, and current status.
