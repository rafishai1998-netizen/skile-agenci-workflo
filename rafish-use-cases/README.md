# RaFish Use Cases

`rafish-use-cases/` is the practical working layer for RaFish. It groups future
Codex/ChatGPT work by business need instead of by the original imported
repository.

Covered areas:

- PrestaShop,
- Allegro/BaseLinker,
- Gmail/Drive,
- graphics and labels,
- marketing/descriptions,
- customer support,
- supplier and product research.

Each subfolder uses the same structure:

- `Cel` - what this area is for.
- `Typowe zadania` - recurring work that Codex/ChatGPT can help with.
- `Przydatne zrodla z sources/` - possible source references, not active
  recommendations.
- `Kandydaci do active/` - items that may be promoted later after review.
- `Status` - current review state.

Stage 3 creates the map only. It does not promote imported content into
`active/`.
