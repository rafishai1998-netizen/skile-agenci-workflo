# Reference

`reference/` is for material that may be useful later but is not ready for
active use.

Do not run workflows, scripts, or automation from `reference/` without a security
review. Imported Claude-only material is not automatically compatible with
Codex or ChatGPT and needs adaptation before promotion.

Stage 3 creates the reference structure only. It does not move imported files
from `sources/`.

Stage 4 adds more reference categories:

- `rube-composio-integrations/` - imported automation skills that depend on
  Rube MCP / Composio.
- `proofpilot-agents/` - future notes for selected ProofPilot doctrine.
- `templates/` - future notes for runnable templates and generated demos.
- `assets/` - future notes for fonts, images, videos, icons, and other media.

No imported files were moved here in stage 4.
