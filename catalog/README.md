# Catalog

This directory provides lightweight indexes over the imported repositories. It
does not replace the source repositories; it points reviewers to the most likely
places to inspect first.

- `skills-index.md` maps likely skill collections and individual skill imports.
- `agents-index.md` maps agent-oriented imports.
- `prompts-index.md` maps prompt and marketing asset imports.
- `repo-type-map.md` classifies the repository into skills, agents, prompts,
  workflows, templates, assets, reference-only material, and generated/cache.
- `usefulness-matrix.md` ranks the most useful paths for RaFish/Codex work.
- `promotion-candidates-stage-4.md` lists the safest first candidates for future
  `active/` promotion.
- `workflows-index.md` records imported workflow material and activation rules.
- `templates-index.md` records runnable template candidates.
- `assets-index.md` records media and asset policy notes.

## Stage 4 Result

The repository should not be divided long-term by original import folder alone.
Use the working categories above, then promote one small reviewed item at a time
into `active/`.
