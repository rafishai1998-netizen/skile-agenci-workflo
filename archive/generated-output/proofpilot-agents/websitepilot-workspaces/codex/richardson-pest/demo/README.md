# Richardson Pest Management Codex Demo

Custom WebsitePilot demo built by Codex for Richardson Pest Management.

## Run

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5187
```

## Verify

```bash
npm run build
npm audit --audit-level=moderate
```

The design uses only Richardson assets copied into `public/assets/rpm/`.
