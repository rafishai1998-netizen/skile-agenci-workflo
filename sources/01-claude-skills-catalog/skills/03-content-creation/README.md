# Content Creation

> Article writing, content strategy, social media distribution, video editing, AI media generation, and quality cleanup for multi-platform publishing.

## When to Use These Skills
- Writing long-form articles, blog posts, or marketing content
- Building a multi-platform content strategy or editorial calendar
- Generating AI images, videos, or media assets
- Cleaning up AI-generated text to sound natural and human

## Start Here
If you're new to this category, start with **article-writing** for long-form content or **content-engine** for multi-platform content production. Use **stop-slop** as a finishing pass on any AI-generated text to remove telltale patterns.

## Skill Relationships
**article-writing** produces long-form content that can be distributed via **crosspost** to social platforms. **content-engine** orchestrates the full pipeline from ideation to publication. **content-strategy-spreadsheet** provides the editorial planning layer that feeds into both writing and distribution. **stop-slop** is a post-processing skill that should be applied to any AI-generated output from the writing skills. **fal-ai-media** and **nano-banana** handle visual asset generation that complements written content. **frontend-slides** transforms content into presentations. **home-service-seo-content** is a vertical specialization combining content creation with SEO best practices.
