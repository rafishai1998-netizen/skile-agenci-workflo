# Session Learning

> Session persistence, continuous learning, instinct tracking, prompt optimization, and documentation lookup for evolving AI capabilities.

## When to Use These Skills
- Preserving context and learnings across sessions
- Evolving skill performance through continuous feedback loops
- Tracking behavioral instincts and optimizing prompts over time
- Looking up documentation for libraries or tools mid-session

## Quick Reference

| Skill | Difficulty | Trigger Phrase | Best For |
|-------|-----------|----------------|----------|
| documentation-lookup | Beginner | `/documentation-lookup` | Finding docs for libraries and tools |
| continuous-learning-v2 | Advanced | `/continuous-learning-v2` | Skill evolution through feedback |
| session-persistence | Intermediate | `/session-persistence` | Saving and restoring session state |
| instinct-tracking | Advanced | `/instinct-tracking` | Tracking behavioral patterns and instincts |
| prompt-optimization | Intermediate | `/prompt-optimization` | Improving prompt effectiveness over time |

## Start Here
If you're new to this category, start with **documentation-lookup** for finding library and tool documentation, or **continuous-learning-v2** for understanding how skills evolve and improve through usage feedback.

## Skill Relationships
**session-persistence** is the infrastructure layer that enables all other session learning skills -- it handles saving and restoring state between sessions. **continuous-learning-v2** builds on persistence to track skill performance and apply improvements over time. **instinct-tracking** monitors behavioral patterns that emerge across sessions, identifying habits that should be reinforced or corrected. **prompt-optimization** uses data from continuous learning and instinct tracking to refine prompts for better results. **documentation-lookup** is the most independent skill -- it provides real-time documentation access that supports any task in any session.
