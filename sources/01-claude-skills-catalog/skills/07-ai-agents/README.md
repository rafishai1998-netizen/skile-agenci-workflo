# AI Agents

> Building AI agents with the Claude API, MCP servers, autonomous loops, agent harnesses, cost optimization, data scrapers, and adaptive learning systems.

## When to Use These Skills
- Building an AI agent or autonomous system using the Claude API
- Creating or configuring MCP (Model Context Protocol) servers
- Implementing agentic loops with tool use, memory, and reasoning
- Optimizing AI agent costs or building data collection pipelines

## Start Here
If you're new to this category, start with **claude-api** for API integration work or **agentic-engineering** for agent design methodology. For MCP-specific work, use **mcp-server**.

## Skill Relationships
**claude-api** is the foundation for all agent skills -- it covers the API primitives that every agent uses. **agentic-engineering** provides the high-level methodology, while **autonomous-agent** implements self-directing loops on top of the API. **mcp-server** and **mcp-tool-builder** handle the tool layer that agents interact with. **agent-memory** and **adaptive-learning-agent** add persistence and learning capabilities. **agent-harness** provides testing infrastructure, and **agent-cost-optimization** addresses production economics. **agent-safety** and **agent-reasoning** are cross-cutting concerns that apply to all agent implementations. **prompt-engineering** feeds into every skill as the interface between human intent and agent behavior.
