# UI/UX Design

> UI/UX design intelligence with 50+ styles, frontend code generation, Lovable.dev prompting, and domain-specific website strategy.

## When to Use These Skills
- Designing user interfaces with specific visual styles or design systems
- Generating frontend code from design specifications
- Creating prompts for Lovable.dev or similar AI design tools
- Planning website architecture and UX for specific industries

## Start Here
If you're new to this category, start with **ui-ux-pro-max** for general design guidance -- it covers 50+ visual styles and can route you to more specific skills. For generating actual frontend code, use **frontend-design**.

## Skill Relationships
**ui-ux-pro-max** is the comprehensive design skill that covers visual styles, layout patterns, accessibility, and design systems. **frontend-design** translates design decisions into code -- it works well as a follow-up to **ui-ux-pro-max** when you need implementation. **lovable-prompting** is a specialized skill for generating effective prompts for the Lovable.dev AI design platform. **property-management-website** is a domain-specific skill that combines UI/UX patterns with property management industry knowledge to produce targeted website strategies.
