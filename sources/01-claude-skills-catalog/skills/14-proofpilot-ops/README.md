# ProofPilot Ops

> ProofPilot agency operations including brand styling, proposals, P&L management, SOWs, Amped onboarding, and ad creation workflows.

## When to Use These Skills
- Creating client deliverables with ProofPilot branding
- Generating proposals, SOWs, or P&L reports for agency clients
- Onboarding new clients to the Amped platform
- Building ad creatives following ProofPilot brand guidelines

## Start Here
If you're new to this category, start with **proofpilot-brand** -- it establishes the brand guidelines and styling that all other ProofPilot skills reference for client deliverables.

## Skill Relationships
**proofpilot-brand** is the foundation -- it defines colors, typography, tone, and visual identity used across all deliverables. **proofpilot-proposal** and **proofpilot-sow** work together in the sales pipeline: proposals convert to SOWs when deals close. **proofpilot-pnl** tracks the financial performance of active engagements. **proofpilot-amped-onboarding** handles the technical setup when bringing new clients onto the Amped platform. **proofpilot-ad-creation** produces ad creatives that follow brand guidelines. **proofpilot-client-report** generates ongoing performance reports that reference P&L data and campaign results from ad creation.
