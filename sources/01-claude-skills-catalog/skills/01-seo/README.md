# SEO

> Comprehensive SEO auditing, optimization, and strategy skills covering technical, content, schema, international, and competitive analysis.

## When to Use These Skills
- Running a full or partial SEO audit on a website
- Optimizing technical infrastructure (sitemaps, schema, hreflang, images)
- Building a content strategy, competitive analysis, or programmatic SEO pipeline
- Preparing geo-targeted or AI-search-optimized content

## Start Here
If you're new to this category, start with **seo** -- it's the most general-purpose and will route you to specialized skills as needed. For a comprehensive site review, use **seo-audit** or **website-seo-audit** directly.

## Skill Relationships
The **seo** skill acts as a router that dispatches to specialist skills based on your request. **seo-audit** and **website-seo-audit** orchestrate multiple specialists (technical, content, schema, images, sitemap) into a single report. The technical skills (**seo-technical**, **seo-schema**, **seo-sitemap**, **seo-images**, **seo-hreflang**) can each be invoked independently for focused analysis. Strategy skills (**seo-plan**, **seo-strategy-spreadsheet**) consume outputs from audit and competitor skills. The programmatic SEO skills (**seo-programmatic**, **programmatic-seo**) build on content and technical foundations to generate pages at scale. Competitor skills (**competitor-seo**, **seo-competitor-pages**) feed into strategy and planning.
