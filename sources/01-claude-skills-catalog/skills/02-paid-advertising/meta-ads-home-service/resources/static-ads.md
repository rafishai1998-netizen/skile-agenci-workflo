# Static Ad Creation for Home Service Businesses

Static image ads are Priority #2 after video ads. They should always be paired
with video ads in the same campaign — Meta's CBO will test both and allocate
budget to winners. Static ads can outperform video (as low as $18 CPL vs $48+ for video).

---

## When to Create Static Ads

- **Always:** Every campaign should have 3-5 static ads alongside 3-5 video ads
- **Client can't film video yet:** Static ads can launch immediately while video is in production
- **Testing new offers:** Quick to create, fast to test new angles
- **Retargeting:** Static ads with strong offers work well for retargeting warm audiences

---

## Static Ad Types That Convert

### Type 1: Before/After Split

The most powerful static ad for home services. Shows the transformation.

**Layout:**
```
┌──────────────────────────────────┐
│  BEFORE          │    AFTER      │
│  [dim/messy      │  [bright/     │
│   photo]         │   clean       │
│                  │   photo]      │
│──────────────────│───────────────│
│  "Your [area]    │  "After [     │
│   right now"     │   BUSINESS]"  │
└──────────────────────────────────┘
```

**Copy guidelines:**
- Use real photos from completed jobs (ask client for their best before/afters)
- Label clearly: "Before" and "After" or use arrow
- Add company logo in corner
- Add trust badges at bottom (licensed, insured, 5-star, X+ reviews)

**Works best for:** Remodeling, painting, landscaping, cleaning, electrical panel upgrades, lighting installs

### Type 2: The Offer Graphic

Clean, bold graphic focused on the offer. Designed to stop the scroll with a dollar amount.

**Layout:**
```
┌──────────────────────────────────┐
│         [COMPANY LOGO]           │
│                                  │
│        $150 OFF                  │
│    Your Next [SERVICE]           │
│                                  │
│  ✓ Licensed & Insured            │
│  ✓ [X] Five-Star Reviews         │
│  ✓ [Key Differentiator]          │
│                                  │
│     Limited Time Only            │
│   [Serving CITY / AREA]          │
└──────────────────────────────────┘
```

**Design rules:**
- Dollar amount is the LARGEST text element
- Use brand colors (get from client's website/logo)
- White or light background performs best (stands out in dark-mode feeds)
- Include 2-3 trust points as checkmarks
- Add "Limited Time" or "This Month Only" for urgency
- City/area name at bottom for local targeting signal

### Type 3: The Job Site Photo

Real, unfiltered photo from a job site with minimal text overlay.

**Layout:**
```
┌──────────────────────────────────┐
│                                  │
│   [Real photo of crew working    │
│    or completed job]             │
│                                  │
│   ┌─────────────────────────┐    │
│   │ Free [SERVICE] Estimate │    │
│   │ Call/Click Today        │    │
│   │ [COMPANY NAME]          │    │
│   └─────────────────────────┘    │
└──────────────────────────────────┘
```

**Why it works:**
- Looks organic in the feed (not "ad-like")
- Real photos build trust
- Minimal text = higher reach (Meta penalizes text-heavy images)
- Shows the business is active and doing real work

### Type 4: The Testimonial Card

Customer review formatted as a visual quote card.

**Layout:**
```
┌──────────────────────────────────┐
│  ★★★★★                          │
│                                  │
│  "[Actual customer review        │
│   text — 2-3 sentences max.      │
│   Focus on specific results.]"   │
│                                  │
│  — [Customer Name], [City]       │
│                                  │
│  ─────────────────────────────   │
│  [COMPANY LOGO]                  │
│  Get Your Free Estimate Today    │
└──────────────────────────────────┘
```

**Guidelines:**
- Pull real reviews from Google Business Profile
- Choose reviews that mention specific results or experiences
- Include the customer's first name and city (makes it feel local)
- Add 5 stars at top
- Keep to 2-3 sentences max

### Type 5: The Problem Awareness Ad

Educates the viewer about a problem they might not know they have.

**Layout:**
```
┌──────────────────────────────────┐
│  ⚠️ [CITY] Homeowners:          │
│                                  │
│  [Photo of common problem —      │
│   old panel, damaged pipe,       │
│   cracked foundation, etc.]      │
│                                  │
│  Is Your [SYSTEM] a              │
│  Safety Hazard?                  │
│                                  │
│  Free Inspection — [COMPANY]     │
│  Licensed • Insured • Local      │
└──────────────────────────────────┘
```

**Works best for:** Electrical (old panels), plumbing (old pipes), HVAC (old units), roofing (storm damage)

## Design Specifications

### Image Sizes
| Placement | Size | Ratio |
|-----------|------|-------|
| Feed (Facebook/Instagram) | 1080x1080px | 1:1 (square) |
| Stories/Reels | 1080x1920px | 9:16 (vertical) |
| Right column | 1200x628px | 1.91:1 (landscape) |

**Recommendation:** Create all ads at 1080x1080 (square) first. This works across Feed and most placements. Create 9:16 versions for Stories/Reels if budget allows.

### Text Rules (Meta's 20% Rule)
- Meta deprioritizes images with more than 20% text coverage
- Keep text minimal — let the image do the work
- Use the ad copy (primary text, headline) for details instead of cramming text on the image
- Logos and small trust badges don't count heavily against the text rule

### Brand Consistency
- Use client's brand colors (pull from their website or logo)
- Include logo on every ad
- Use consistent fonts across all creatives
- If client has a branded vehicle, include it when possible

## Creating Static Ads Without a Designer

### Option 1: Canva (Free/Pro)
Best for non-designers. Use templates and customize.
1. Search "Facebook Ad" templates
2. Pick a clean template
3. Replace with client's photos, logo, colors
4. Export at 1080x1080 PNG

### Option 2: AI Image Generation (Advanced Only)
For experienced ad buyers only. Test AI-generated images of homes, work results.
- Steve Hunsaker's warning: "Don't start here. Master the basics first."
- AI images can work for before/after mockups or lifestyle imagery
- Always pair with real photos — never use AI images exclusively

### Option 3: Client Photos + Text Overlay
Simplest approach. Take the client's best job photos and add minimal text.
- Use iPhone Photos app or CapCut for simple text overlays
- Keep text to: Company name + Offer + City
- This "raw" look often outperforms polished designs in testing

## Static Ad Testing Strategy

1. **Launch 3-5 static ads alongside 3-5 video ads** in one CBO campaign
2. **Meta's algorithm tests all creatives** and shifts budget to winners
3. **Review after 3-5 days:** Which statics are getting spend? Which have lowest CPL?
4. **Kill obvious losers** (high CPL, zero engagement after sufficient spend)
5. **Create variations of winners:** Same layout, different photo. Same photo, different offer.
6. **Refresh every 2-4 weeks** at higher spend levels to combat creative fatigue

### What to Test
| Variable | Example Variations |
|----------|-------------------|
| Image type | Before/after vs. job site vs. offer graphic |
| Offer | "$150 off" vs. "Free inspection" vs. "20% off" |
| Color scheme | Dark background vs. light background vs. brand colors |
| Text amount | Minimal (logo + offer) vs. detailed (3 trust points) |
| Photo style | Professional photography vs. iPhone photos |

## Pairing Static + Video Ads

The ideal campaign creative mix:

| Campaign Budget | Video Ads | Static Ads | Total Creatives |
|----------------|-----------|------------|-----------------|
| $40-50/day | 3 | 2-3 | 5-6 |
| $100-150/day | 4-5 | 3-4 | 7-9 |
| $200+/day | 5-8 | 4-5 | 9-13 |

**Why this works:**
- Video ads build trust and brand awareness (owner on camera)
- Static ads catch people who don't watch video (scrollers, at work, sound off)
- Meta shows different creatives to different users based on their behavior
- Some users convert after seeing a video first, then a static ad (nurture effect)
- The combination gives the algorithm more data points to optimize
