# Web Scraping

> Firecrawl-powered suite for web scraping, search, crawling, site mapping, downloading, autonomous extraction, and browser automation.

## When to Use These Skills
- Scraping content from websites for research or data collection
- Searching the web programmatically for specific information
- Mapping a site's structure or downloading entire pages
- Automating browser interactions for data extraction

| Skill | Difficulty | Trigger Phrase | Best For |

## Start Here
If you're new to this category, start with **firecrawl** for general-purpose scraping or **firecrawl-search** for web search. These two cover the most common use cases and are the simplest to invoke.

## Skill Relationships
**firecrawl** is the general-purpose entry point that can handle most scraping needs. For targeted tasks, the specialized skills offer more control: **firecrawl-scrape** extracts content from a single URL, while **firecrawl-crawl** follows links recursively across a site. **firecrawl-map** is useful as a precursor to crawling -- it discovers the site structure first so you can target specific sections. **firecrawl-search** operates independently as a web search tool. **firecrawl-download** saves full pages locally. **firecrawl-agent** and **firecrawl-browser** are the most advanced options, adding AI-driven autonomous extraction and interactive browser control respectively.
