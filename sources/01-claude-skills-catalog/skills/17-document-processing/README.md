# Document Processing

> Process, convert, translate, and manipulate documents and media files

## When to Use These Skills
- Converting or processing PDFs, DOCX, XLSX files
- Translating visa or legal documents with bilingual output
- Working with video/audio — ingesting, indexing, editing, generating
- Posting to or reading from X/Twitter programmatically

## Quick Reference

| Skill | Difficulty | Trigger Phrase | Best For |
|-------|-----------|----------------|----------|
| nutrient-document-processing | Intermediate | "process PDF", "convert document" | PDF/DOCX/XLSX processing, OCR, redaction |
| visa-doc-translate | Beginner | "translate visa document" | Bilingual PDF creation from document images |
| videodb | Advanced | "process video", "index video" | Video/audio ingestion, search, editing |
