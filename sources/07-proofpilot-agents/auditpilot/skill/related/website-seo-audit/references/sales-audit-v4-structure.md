# Sales Audit v4 Structure (Matthew-approved)

Proven on AP Pest Control April 2026 across 4 iterations. This is the canonical
structure for ProofPilot strategic sales audits when the goal is to pitch
services and move the deal forward (NOT a technical post-onboarding QA).

## Section Order

1. **Where You Stand** — Scorecard and verdict, direct and sales-friendly. No
   "CEO talking point" boxes anywhere in the doc. Ever.

2. **The Ranking Reality** — THIS IS THE LEAD FINDING. Live SERP data for
   10-25 core local queries showing the client ranks for zero (or near zero).
   Include the indexing stat (sitemap pages vs actually indexed by Google).
   Make it visceral.

3. **Who Is Outranking You** — Two parts:
   - 3a. Competitive landscape table (7+ real competitors, services, content
     footprint, why they rank).
   - 3b. **What They Are Actually Ranking For** — Per-competitor keyword tables
     with real DataForSEO volumes. Position | Keyword | Monthly Searches.
     Closes with "Client: ranking for 0 of N core keywords" in red. This
     section creates the "I want THAT" feeling. Mandatory.

4. **Where You Should Be Ranking** — Three phases:
   - Core Money Keywords (with DataForSEO volumes and CPCs)
   - Long-Tail City + Service Combinations
   - Informational Blog content (point at competitor blog wins as proof)

5. **Services You Should Be Selling** — Pure logic-driven. Look at owner bio
   (years experience, certifications, prior roles) and identify services they
   clearly offer in real life but have NO page for. Frame as "there's no way
   you're turning down those calls, you just don't have pages selling them."

6. **Red Flags** — Schema findings live HERE as a SUBSECTION, not their own
   section. Explain in plain English ("your developer copy-pasted a template
   from X and never updated it") with a what-it-says-vs-what-it-should-say
   table. Other subsections: location page template duplication, NAP drift,
   gmail address, dead social icons, empty blog, form issues.

7. **The Funnel Math** — NOT Google Ads equivalent. NOT monthly spend
   equivalent. See funnel-math format below.

8. **30-Day Quick Wins** — Week-by-week breakdown.

9. **The ProofPilot Growth Playbook** — Sell the dream. Six pillars:
   1. Website built to rank
   2. Territory Domination SEO (monthly cadence: 4 service pages, 4 location
      pages, 4 blog posts, 8-12 backlinks)
   3. AEO / AI Search Optimization (ChatGPT, Gemini, Perplexity, AI Overviews)
   4. Reddit, Nextdoor, community groups (point out Reddit currently outranks
      most local sites in their market)
   5. Paid media as the accelerator
   6. Lead tracking and reporting

10. **Where We Take You In 12 Months** — Side-by-side from→to closing list.
    Examples: 0/24 keywords ranking → top 5, 5 indexed pages → 100+, 0 blog
    posts → 48, invisible in AI → cited in ChatGPT/Gemini/Perplexity, 0
    backlinks → 100+.

## Funnel Math Format (Section 7)

Three scenarios: **Conservative / Realistic / Aggressive**.

Walk down the funnel step by step in a single table:

| Step | Conservative | Realistic | Aggressive |
|---|---|---|---|
| Avg ranking position | #8-10 | #4-5 | #2-3 |
| CTR | 3% | 7% | 15% |
| Monthly organic clicks | (volume × CTR) | | |
| Visitor-to-lead | 4% | 4% | 5% |
| Monthly leads | | | |
| Close rate | 35% | 35% | 40% |
| Monthly new customers | | | |
| Annual new customers | | | |

Then a Revenue table:

| | Conservative | Realistic | Aggressive |
|---|---|---|---|
| Year 1 residential | (customers × Y1 value) | | |
| Commercial accounts | | | |
| Commercial revenue | | | |
| **Total Year 1** | | | |
| **3-year total impact** | (customers × LTV) | | |

**Always include a "Why These Numbers Are Real" section sourcing every
assumption**: their own pricing from their own site, their own review rating,
real DataForSEO keyword volumes, published CTR studies. This is what makes
the math feel grounded instead of made up.

Close with a "Cost of Waiting" paragraph.

## Hard Rules

- NO "CEO talking point" callout boxes
- NO em dashes
- NO Google Ads equivalent / monthly ad spend equivalent for revenue impact
- Schema findings explained in plain English, never as "schema validation"
- Show what's going RIGHT before calling out mistakes
- Lead with ranking reality + competitor keyword wins
- Use logic to identify missing service pages from owner bio
- Position ProofPilot as a scaling growth partner across web/SEO/AEO/Reddit/paid
- Apply full ProofPilot brand (Bebas Neue, eyebrow+title, tables, callouts)
- Save to Proposals folder in Drive
