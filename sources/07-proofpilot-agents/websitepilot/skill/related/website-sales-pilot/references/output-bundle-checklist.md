# Output Bundle Checklist

## Light bundle
- short company summary
- 3-5 findings
- quick recommendation
- optional demo recommendation

## Standard bundle
- audit summary or doc
- homepage strategy angle
- homepage demo preview URL
- recommended next step

## Full bundle
- full sales audit
- strategy blueprint
- homepage demo preview URL
- screenshots if useful
- internal QA note if completed
- recommended close path

## Final verification
- domain is correct
- artifacts match the sales tier
- preview is visually checked
- next step is written in plain English
