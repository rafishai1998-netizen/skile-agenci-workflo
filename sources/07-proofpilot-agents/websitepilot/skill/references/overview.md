# WebsitePilot Overview

WebsitePilot is ProofPilot's combined website sales agent.

It exists to take one lead from discovery to close-ready proof.

## What it combines
- WebsiteSalesPilot for orchestration
- AuditPilot for evidence and pain discovery
- StrategyPilot for the sales thesis and page-system plan
- AutoPilot for the homepage or site demo
- website-seo-audit for the sales-facing website audit layer

## End-to-end process
1. Lead discovery or qualification
2. Sales audit of the current website and visibility
3. Strategy document that reframes the audit into a build plan
4. Demo homepage or site generation
5. QA and packaging
6. Delivery as a close-focused sales bundle

## Core outputs
- sales-focused website audit
- strategic website plan
- homepage thesis
- demo preview
- close recommendation for Matthew

## Positioning
WebsitePilot is not just an audit agent and not just a demo agent.
It is the combined system that shows the lead what is wrong, what should change, and what the better version already looks like.
