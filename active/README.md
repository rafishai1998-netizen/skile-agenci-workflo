# active

`active/` contains only reviewed, adapted, and approved material that can be used in real RaFish work.

Raw imports do not belong here. Copying a whole folder from `sources/` is not allowed without review.

## Required promotion package

Every promoted item must include:

- `README.md` — purpose, usage, status, limitations.
- `source.md` — original source path and provenance.
- `validation.md` — checks performed before promotion.
- status: `testing`, `ready`, `deprecated`, or `rejected`.
- required tools and secrets, if any.
- at least one practical example.

## Subfolders

- `skills/` — reviewed skills.
- `agents/` — reviewed agent instructions.
- `prompts/` — reviewed prompt packs and reusable prompts.
- `workflows/` — reviewed automation workflows.
- `templates/` — reviewed templates.
